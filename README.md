# Модель для предсказания вероятности клика (CTR) на рекламное объявление

## Описание проекта

Данный проект представляет собой разработку модели для прогнозирования кликабельности рекламных объявлений на платформе «Advandex».

## Установка

1.  Клонирование репозитория:
    ```bash
    git clone https://github.com/username/project.git
    cd project
    ```
2.  Создание виртуального окружения:
    ```bash
    python -m venv venv
    source venv/bin/activate
    ```
3.  Установка зависимостей:
    ```bash
    pip install -r requirements.txt
    ```
4.  Настройка базы данных:
    Создать файл .env с параметрами подключения к базе данных.

5.  Запуск Jupyter Notebook:
    Открой файл project.ipynb и выполни ядро («Kernel → Restart & Run All»), предварительно настроив подключение к базе данных через файл .env.

## Структура проекта

- `.gitignore`: Файл с перечнем исключаемых из git-трекинга элементов.
- `PROJECT.ipynb`: Jupyter notebook с кодом и результатом анализа.
- `requirements.txt`: Перечень зависимостей проекта.
- `README.md`: Руководство по проекту в виде markdown-документа.

## Используемые технологии

- Python: основной язык программирования.
- Pandas: библиотека для обработки и анализа данных.
- Matplotlib / Seaborn: инструменты для визуализации данных.
- Scikit-learn (sklearn): основная библиотека для моделирования, предобработки данных и оценки качества моделей.
- Phik: библиотека для анализа корреляций и статистической значимости между переменными.
- Category Encoders: библиотека для расширенного кодирования категориальных признаков (например, Target Encoding).
- Joblib: библиотека для сериализации и сохранения обученных моделей.
